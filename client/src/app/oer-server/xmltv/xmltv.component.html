XmlTV

XmlTV is a standardized format for TV program data. It is used by many TV applications to display program information.
Oerc supports exporting the program data in the XmlTV format via HTTP api and CLI command xmltv:

{{ exampleXmlTvApiUrl }}

By clicking on the following buttons you can download the XmlTV file for the selected time period.
NOTE: You can adjust the date range also directly in the URL.
